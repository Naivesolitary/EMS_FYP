# Evenera – Event Management and Booking System

A full-stack web application for managing events, bookings, and user registrations.

## Features
- Event creation and management
- User registration and booking system
- Role-based authentication (admin, organizer, user)
- REST API backend

## Tech Stack
Frontend: React.js, Tailwind CSS  
Backend: Node.js, Express.js  
Database: MySQL  

## Purpose
Final year project built to demonstrate full-stack development and system design skills.
